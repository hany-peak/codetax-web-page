# 코드세무회계 메인 웹사이트 설계

## Context

현재 `index.html`은 채용공고 전용 페이지. 세무회계 서비스를 소개하는 **메인 랜딩 웹사이트**가 필요하다. rodemtax.com을 참고하되, 히어로 섹션에 **3D 인터랙티브 캐릭터**(마우스 추적)를 넣어 차별화한다.

---

## 1. 3D 히어로 섹션 (핵심)

### 기술: Spline (@splinetool/runtime)

- **Spline**에서 3D 캐릭터 씬을 제작 → `.splinecode`로 export
- `@splinetool/runtime` (~70KB gzipped)으로 웹에 임베드
- Spline의 내장 **"Look At" / "Mouse Hover"** 트리거로 마우스 추적 구현

### 캐릭터 컨셉

- **귀여운 둥근 캐릭터들** (다양한 색상, 표정)이 모여 있는 구성
- 마우스 이동 시:
  - 캐릭터들의 **눈이 커서를 따라감** (Look At)
  - 캐릭터 **몸/머리가 마우스 방향으로 살짝 회전** (로봇 레퍼런스처럼)
- 세무회계 브랜드와 연결: 캐릭터들이 계산기, 서류, 돋보기 등 소품을 들고 있으면 좋음
- 배경: 밝은 톤 (기존 사이트 컬러 `#F8F7F4` 계열)

### 성능 최적화

- **Lazy load**: `IntersectionObserver`로 뷰포트 진입 시 Spline 로드
- **Static fallback**: WebGL 미지원 기기에는 캐릭터 스냅샷 이미지 표시
- **모바일**: 터치 이벤트 → 마우스 추적 대체, 또는 자동 애니메이션 루프

### 히어로 레이아웃

```
┌─────────────────────────────────────────────────┐
│  Nav: 로고 | 서비스 | 회사소개 | 상담신청 버튼    │
├─────────────────────────────────────────────────┤
│                                                 │
│   ┌──────────────┐  ┌────────────────────────┐  │
│   │  헤드카피     │  │                        │  │
│   │  "AI 시대의   │  │   [3D Spline Canvas]   │  │
│   │   똑똑한      │  │   캐릭터들이 마우스를    │  │
│   │   세무파트너"  │  │   따라 움직임           │  │
│   │              │  │                        │  │
│   │  서브카피     │  │                        │  │
│   │  + CTA 버튼  │  │                        │  │
```

- 데스크톱: 좌측 텍스트 + 우측 3D 캔버스 (50/50)
- 모바일: 상단 텍스트 → 하단 3D 캔버스 (스택 레이아웃)

## 3. 기술 스택

| 영역 | 선택 | 이유 |
|------|------|------|
| 마크업 | HTML5 단일 파일 | 기존 프로젝트 패턴 유지 |
| 스타일 | Embedded CSS + CSS Variables | 기존 디자인 토큰 재사용 |
| 3D | Spline (@splinetool/runtime) | 마우스 추적 내장, 낮은 개발 복잡도 |
| 애니메이션 | CSS @keyframes + IntersectionObserver | 스크롤 진입 시 fadeUp |
| 폰트 | Noto Sans KR + DM Serif Display | 기존 폰트 스택 유지 |
| 아이콘 | Inline SVG | 외부 의존성 없음 |

## 7. Spline 씬 제작 가이드

Spline에서 직접 만들어야 할 것들:

1. **캐릭터 모델링**: 둥근 머리 + 큰 눈 + 작은 몸체, 5-8개 캐릭터
2. **색상**: 빨강, 파랑, 초록, 분홍, 보라, 노랑 등 다양한 컬러
3. **소품**: 계산기, 서류, 돋보기, 연필 (세무 컨셉)
4. **인터랙션 설정**:
   - Event: "Mouse Hover" on canvas
   - Action: "Look At" cursor (각 캐릭터의 눈/머리에 적용)
   - Action: "Rotate" toward cursor (몸체에 살짝 적용, damping 0.1)
5. **카메라**: 정면, 약간 위에서 내려다보는 앵글
6. **조명**: 따뜻한 톤, 소프트 쉐도우

## 8. 검증 방법

1. **로컬 서버 실행**: `npx serve .` 또는 Live Server로 main.html 열기
2. **3D 인터랙션 테스트**: 마우스 움직임 → 캐릭터 눈/몸 추적 확인
3. **반응형 테스트**: 브라우저 DevTools에서 모바일/태블릿 뷰 확인
4. **폴백 테스트**: WebGL 비활성화 후 폴백 이미지 표시 확인
5. **성능 테스트**: Lighthouse로 LCP, FCP 측정 (목표: LCP < 2.5s)
6. **크로스 브라우저**: Chrome, Safari, Firefox 에서 동작 확인
